# AR TECH Solutions

Full-stack SaaS website for AR TECH Solutions — digital consulting and hotel-tech platform.

## Quick Start

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm start
```

### Environment
Create `server/.env`:
```
MONGODB_URI=mongodb://localhost:27017/artech
PORT=5000
```

## URLs
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
- Admin Dashboard: http://localhost:3000/admin/leads

## Tech Stack
- **Frontend**: React 18, Tailwind CSS, Framer Motion, React Router v6
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Forms**: express-validator, react-hot-toast
